# Список пользователей 

## Название проекта

**UserList Viewer — динамическая загрузка данных пользователей**

## Цель

Приложение демонстрирует простую работу с внешним API: загрузку данных, их отображение в таблице и обработку ошибок. Код подходит как пример для изучения основ web‑разработки и взаимодействия с `fetch()`.

## Что делает приложение

* Отправляет запрос к `https://jsonplaceholder.typicode.com/users`.
* Получает список пользователей в формате JSON.
* Отображает данные в виде таблицы: ID, имя пользователя, email и город.
* Показывает индикатор загрузки.
* Сообщает об ошибке при проблемах с сетью.
* Позволяет вручную обновить данные кнопкой «Обновить данные».

## Дополнительная информация
Приложение использует минимальный набор веб-технологий: HTML, CSS и JavaScript без библиотек. Такой подход позволяет полностью понимать, как обрабатывается запрос, формируется таблица и управляются состояния.

### Ключевые особенности реализации
- Асинхронная функция `loadUsers()` с `async/await`.
- Обработка ошибок через `try...catch`.
- Динамическое создание элементов таблицы.
- Простая модульность: логику легко масштабировать.
- Подходит как основа для SPA или мини‑панели администратора.
